## 1. 方案概述

### 1.1 背景

为部署在云端的 Agent 提供安全的 Bash 工具执行环境，需要实现：
- **用户数据持久化**：Agent 执行产生的文件和状态需要跨请求保留
- **用户数据隔离**：不同 Agent/用户的数据互不可见

### 1.2 解决方案

采用 **Zeroboot + 外部 Gateway** 的分层架构：

| 组件 | 职责 |
|------|------|
| Zeroboot (原生) | 提供基于 KVM/Firecracker 的轻量级沙盒执行环境 |
| Sandbox Gateway | 封装 Zeroboot，提供用户管理、数据持久化、认证鉴权 |

### 1.3 优势

- Zeroboot 升级无感知，仅替换二进制文件
- 业务逻辑与执行引擎解耦
- 故障隔离，Gateway 挂了不影响 Zeroboot 原生功能

### 2.3 用户数据隔离

```
/var/lib/zeroboot/users/
├── {hash(user_id_a)}/           # Agent A 的数据
│   ├── workspace/
│   │   ├── file1.txt
│   │   └── .env
│   ├── home/
│   │   └── .bashrc
│   └── .metadata/
│       ├── created_at
│       └── last_access
│
└── {hash(user_id_b)}/           # Agent B 的数据 (完全隔离)
    ├── workspace/
    └── ...

权限配置:
- 每个用户目录: 0700 (仅 zeroboot 用户可访问)
- Gateway 以 zeroboot 用户运行
- 不同用户的 UID 不同，无法互相访问
```

## 4. 部署步骤

### 4.1 系统准备

```bash
# 1. 更新系统
sudo apt update && sudo apt upgrade -y

# 2. 安装基础依赖
sudo apt install -y \
    curl \
    wget \
    git \
    build-essential \
    libssl-dev \
    pkg-config

# 3. 安装 KVM 相关包
sudo apt install -y \
    qemu-kvm \
    libvirt-daemon-system \
    libvirt-clients \
    bridge-utils \
    virtinst

# 4. 添加当前用户到 kvm/libvirt 组 (可选，生产环境不建议)
# sudo usermod -aG kvm,libvirt $USER

# 5. 验证 KVM
ls -la /dev/kvm
# 应显示: crw-rw---- 1 root kvm ...
```

### 4.2 创建系统用户

```bash
# 1. 创建 zeroboot 系统用户
sudo useradd \
    --system \
    --no-create-home \
    --shell /bin/false \
    --comment "Zeroboot Service" \
    zeroboot

# 2. 创建数据目录
sudo mkdir -p /var/lib/zeroboot/{templates,users,logs}
sudo mkdir -p /var/run/zeroboot

# 3. 设置权限
sudo chown -R zeroboot:zeroboot /var/lib/zeroboot
sudo chown -R zeroboot:zeroboot /var/run/zeroboot
sudo chmod 700 /var/lib/zeroboot/users
```

### 4.3 安装 Zeroboot 原生服务

```bash
# 1. 下载 Zeroboot (请替换为实际版本号)
ZERBOOT_VERSION="v1.2.0"
cd /tmp
wget https://github.com/anthropics/zeroboot/releases/download/${ZERBOOT_VERSION}/zeroboot-${ZERBOOT_VERSION}-linux-amd64.tar.gz

# 2. 解压安装
sudo tar -xzf zeroboot-${ZERBOOT_VERSION}-linux-amd64.tar.gz -C /opt/zeroboot/
sudo ln -sf /opt/zeroboot/zeroboot /usr/local/bin/zeroboot

# 3. 验证安装
zeroboot --version
```

### 4.4 创建 VM 模板

```bash
# 1. 准备内核和根文件系统
# 下载预构建的 rootfs (推荐) 或自行构建
cd /var/lib/zeroboot/templates
sudo -u zeroboot mkdir -p python

# 下载示例 rootfs (请从官方获取)
# wget https://github.com/anthropics/zeroboot/releases/download/v1.0.0/python-rootfs.tar.gz

# 2. 创建模板快照 (需要内核镜像)
KERNEL_PATH="/boot/vmlinuz-$(uname -r)"
ROOTFS_PATH="./python/rootfs.ext4"

# 如果有 rootfs，执行:
# sudo -u zeroboot zeroboot template ${KERNEL_PATH} ${ROOTFS_PATH} ./python

# 3. 检查模板状态
ls -la /var/lib/zeroboot/templates/python/snapshot/
# 应包含: mem, vmstate
```

### 4.6 配置 Systemd 服务

#### Zeroboot 原生服务

```bash
# 创建 zeroboot 服务文件
sudo tee /etc/systemd/system/zeroboot.service << 'EOF'
[Unit]
Description=Zeroboot Sandboxed Execution Engine
After=network.target
Wants=network.target

[Service]
Type=simple
User=zeroboot
Group=zeroboot
ExecStart=/usr/local/bin/zeroboot serve /var/lib/zeroboot/templates 8081
Restart=always
RestartSec=5
StandardOutput=journal
StandardError=journal

# 安全加固
NoNewPrivileges=true
ProtectSystem=strict
ProtectHome=true
ReadOnlyPaths=/var/lib/zeroboot/templates
ReadOnlyPaths=/boot
ReadOnlyPaths=/usr
ReadOnlyPaths=/lib
ReadOnlyPaths=/lib64
PrivateTmp=true
ProtectKernelTunables=true
ProtectKernelModules=true
ProtectControlGroups=true
RestrictNamespaces=true
LockPersonality=true
MemoryDenyWriteExecute=false

# 日志
SyslogIdentifier=zeroboot

[Install]
WantedBy=multi-user.target
EOF
```

#### Sandbox Gateway 服务

```bash
# 创建 gateway 服务文件
sudo tee /etc/systemd/system/sandbox-gateway.service << 'EOF'
[Unit]
Description=Sandbox Gateway - User Management & Data Persistence
After=network.target zeroboot.service
Wants=zeroboot.service

[Service]
Type=simple
User=zeroboot
Group=zeroboot
WorkingDirectory=/opt/sandbox-gateway
ExecStart=/usr/bin/node /opt/sandbox-gateway/src/index.js
Restart=always
RestartSec=5
StandardOutput=journal
StandardError=journal
Environment=NODE_ENV=production

# 安全加固
NoNewPrivileges=true
ProtectSystem=strict
ProtectHome=true
ReadOnlyPaths=/opt/sandbox-gateway
PrivateTmp=true
ProtectKernelTunables=true
ProtectKernelModules=true
ProtectControlGroups=true
RestrictNamespaces=true

# 资源限制
LimitNOFILE=65536
MemoryMax=1G

# 日志
SyslogIdentifier=sandbox-gateway

[Install]
WantedBy=multi-user.target
EOF
```

## 7. 安全加固

### 7.1 网络隔离

```bash
# 仅允许内网访问 Zeroboot
sudo ufw allow from 10.0.0.0/8 to any port 8081
sudo ufw allow from 172.16.0.0/12 to any port 8081
sudo ufw allow from 192.168.0.0/16 to any port 8081

# Gateway 对外开放 (或仅允许 Agent Server IP)
sudo ufw allow 8080/tcp

# 启用防火墙
sudo ufw enable
```

### 7.2 代码执行限制

```javascript
// src/services/zeroboot.js

const CODE_RESTRICTIONS = {
  // 禁止的危险命令
  forbidden_patterns: [
    /rm\s+-rf\s+\/(?!workspace)/,           // 禁止根目录删除
    /dd\s+/,                                  // 禁止直接磁盘操作
    /mkfs/,                                   // 禁止格式化
    /:(){/:|:&};:/,                          // 禁止 fork 炸弹
    /curl\s+http.*\|sh/,                      // 禁止管道执行远程脚本
    /wget.*\|sh/,
  ],
  
  // 允许的路径
  allowed_paths: [
    '/workspace',
    '/tmp',
    '/home',
  ],
  
  // 最大文件大小 (字节)
  max_file_size: 100 * 1024 * 1024,  // 100MB
  
  // 单次操作最大文件数
  max_file_operations: 1000,
};

function validateCode(code) {
  for (const pattern of CODE_RESTRICTIONS.forbidden_patterns) {
    if (pattern.test(code)) {
      throw new Error(`Code contains forbidden pattern: ${pattern}`);
    }
  }
  return true;
}
```

### 7.3 用户配额

```javascript
// src/services/storage.js

const USER_QUOTAS = {
  default: {
    storage_bytes: 1 * 1024 * 1024 * 1024,   // 1GB
    max_files: 10000,
    max_execution_time: 300,                  // 5分钟
    rate_limit: 100,                          // 每分钟
  },
  
  premium: {
    storage_bytes: 10 * 1024 * 1024 * 1024,  // 10GB
    max_files: 100000,
    max_execution_time: 1800,                // 30分钟
    rate_limit: 1000,
  },
};

function checkQuota(userId, usage) {
  const quota = USER_QUOTAS.default; // 可从数据库获取用户级别
  
  if (usage.storage_bytes > quota.storage_bytes) {
    throw new Error('Storage quota exceeded');
  }
  
  if (usage.max_files > quota.max_files) {
    throw new Error('File count quota exceeded');
  }
  
  return true;
}
```

### src/services/zeroboot.js

```javascript
const axios = require('axios');
const { v4: uuidv4 } = require('uuid');
const { config } = require('../config');
const { storageService } = require('./storage');
const { validateCode } = require('../utils/security');
const { logger } = require('../utils/logger');

const zerobootClient = axios.create({
  baseURL: config.zeroboot.url,
  timeout: config.zeroboot.timeout,
});

async function executeCode({ code, language, timeout, workdir, userId }) {
  const startTime = Date.now();
  const execId = uuidv4();
  
  logger.info(`Executing code for user ${userId}`, { execId });
  
  // Validate code
  validateCode(code);
  
  // Validate language
  if (!config.security.allowedLanguages.includes(language)) {
    throw Object.assign(new Error(`Unsupported language: ${language}`), {
      status: 400,
      code: 'INVALID_LANGUAGE',
    });
  }
  
  // Prepare user data
  const tempDir = await storageService.prepareForExec(userId, execId);
  
  try {
    // Wrap code to work in user's workspace
    const wrappedCode = workdir 
      ? `cd ${workdir}\n${code}`
      : `cd /workspace\n${code}`;
    
    // Call Zeroboot
    const response = await zerobootClient.post('/v1/exec', {
      code: wrappedCode,
      language,
      timeout_seconds: timeout || config.security.maxExecutionTime,
    });
    
    // Sync results back
    await storageService.saveAfterExec(userId, tempDir);
    
    const execTime = Date.now() - startTime;
    
    logger.info(`Execution completed`, { execId, execTime });
    
    return {
      id: execId,
      stdout: response.data.stdout || '',
      stderr: response.data.stderr || '',
      exit_code: response.data.exit_code || 0,
      exec_time_ms: execTime,
    };
  } catch (error) {
    logger.error(`Execution failed`, { execId, error: error.message });
    
    // Try to save partial results
    try {
      await storageService.saveAfterExec(userId, tempDir);
    } catch (syncError) {
      logger.error(`Failed to sync partial results`, { execId });
    }
    
    if (error.code === 'ECONNREFUSED') {
      throw Object.assign(new Error('Execution engine unavailable'), {
        status: 503,
        code: 'ENGINE_UNAVAILABLE',
      });
    }
    
    if (error.response?.data?.stderr?.includes('timed out')) {
      throw Object.assign(new Error('Execution timed out'), {
        status: 408,
        code: 'TIMEOUT',
      });
    }
    
    throw error;
  } finally {
    // Cleanup temp directory
    await storageService.cleanupTemp(execId);
  }
}

async function executeBatch(executions, userId) {
  const results = [];
  
  for (const exec of executions) {
    const result = await executeCode({
      ...exec,
      userId,
    });
    results.push(result);
  }
  
  return results;
}

async function checkZeroboot() {
  try {
    await zerobootClient.get('/health', { timeout: 5000 });
    return true;
  } catch {
    return false;
  }
}

module.exports = {
  executeCode,
  executeBatch,
  checkZeroboot,
};
```

### src/services/storage.js

```javascript
const fs = require('fs').promises;
const path = require('path');
const crypto = require('crypto');
const { config } = require('../config');
const { logger } = require('../utils/logger');

class StorageService {
  constructor() {
    this.userDataRoot = config.storage.userDataRoot;
    this.tempDataRoot = config.storage.tempDataRoot;
  }
  
  // Hash user ID for directory name (privacy)
  hashUserId(userId) {
    return crypto.createHash('sha256').update(userId).digest('hex').substring(0, 16);
  }
  
  getUserDir(userId) {
    return path.join(this.userDataRoot, this.hashUserId(userId));
  }
  
  getTempDir(execId) {
    return path.join(this.tempDataRoot, `exec_${execId}`);
  }
  
  async prepareForExec(userId, execId) {
    const userDir = this.getUserDir(userId);
    const tempDir = this.getTempDir(execId);
    
    // Create temp directory
    await fs.mkdir(tempDir, { recursive: true });
    
    // Copy user data to temp (only workspace)
    const userWorkspace = path.join(userDir, 'workspace');
    const tempWorkspace = path.join(tempDir, 'workspace');
    
    try {
      await fs.access(userWorkspace);
      await this.copyDir(userWorkspace, tempWorkspace);
    } catch {
      // Create empty workspace if doesn't exist
      await fs.mkdir(tempWorkspace, { recursive: true });
    }
    
    logger.debug(`Prepared temp for exec`, { userId, execId, tempDir });
    
    return tempDir;
  }
  
  async saveAfterExec(userId, tempDir) {
    const userDir = this.getUserDir(userId);
    const tempWorkspace = path.join(tempDir, 'workspace');
    const userWorkspace = path.join(userDir, 'workspace');
    
    try {
      await fs.access(tempWorkspace);
      
      // Create user directory if doesn't exist
      await fs.mkdir(userWorkspace, { recursive: true });
      
      // Sync back changes
      await this.syncDir(tempWorkspace, userWorkspace);
    } catch (error) {
      if (error.code !== 'ENOENT') {
        throw error;
      }
    }
    
    // Update last access
    await this.updateMetadata(userId);
  }
  
  async cleanupTemp(execId) {
    const tempDir = this.getTempDir(execId);
    
    try {
      await fs.rm(tempDir, { recursive: true, force: true });
    } catch (error) {
      logger.warn(`Failed to cleanup temp`, { execId, error: error.message });
    }
  }
  
  async copyDir(src, dest) {
    await fs.mkdir(dest, { recursive: true });
    const entries = await fs.readdir(src, { withFileTypes: true });
    
    for (const entry of entries) {
      const srcPath = path.join(src, entry.name);
      const destPath = path.join(dest, entry.name);
      
      if (entry.isDirectory()) {
        await this.copyDir(srcPath, destPath);
      } else {
        await fs.copyFile(srcPath, destPath);
      }
    }
  }
  
  async syncDir(src, dest) {
    await fs.mkdir(dest, { recursive: true });
    const entries = await fs.readdir(src, { withFileTypes: true });
    
    for (const entry of entries) {
      const srcPath = path.join(src, entry.name);
      const destPath = path.join(dest, entry.name);
      
      if (entry.isDirectory()) {
        await this.syncDir(srcPath, destPath);
      } else {
        const srcStat = await fs.stat(srcPath);
        let shouldCopy = true;
        
        try {
          const destStat = await fs.stat(destPath);
          // Only copy if source is newer
          shouldCopy = srcStat.mtimeMs > destStat.mtimeMs;
        } catch {
          // Destination doesn't exist
        }
        
        if (shouldCopy) {
          await fs.copyFile(srcPath, destPath);
        }
      }
    }
  }
  
  async getUserInfo(userId) {
    const userDir = this.getUserDir(userId);
    
    try {
      const stats = await fs.stat(userDir);
      const size = await this.getDirSize(path.join(userDir, 'workspace'));
      const metadata = await this.getMetadata(userId);
      
      return {
        user_id: userId,
        created_at: stats.birthtime,
        storage_used_bytes: size,
        last_activity: metadata.last_access,
        workspace_files: await this.countFiles(path.join(userDir, 'workspace')),
      };
    } catch (error) {
      if (error.code === 'ENOENT') {
        return {
          user_id: userId,
          created_at: null,
          storage_used_bytes: 0,
          last_activity: null,
          workspace_files: 0,
        };
      }
      throw error;
    }
  }
  
  async deleteUser(userId) {
    const userDir = this.getUserDir(userId);
    await fs.rm(userDir, { recursive: true, force: true });
    logger.info(`User deleted`, { userId });
  }
  
  async listUsers() {
    try {
      const entries = await fs.readdir(this.userDataRoot);
      return entries.map(dir => ({
        user_hash: dir,
        created_at: null, // Would need to read metadata
      }));
    } catch {
      return [];
    }
  }
  
  async getDirSize(dirPath) {
    let size = 0;
    
    try {
      const entries = await fs.readdir(dirPath, { withFileTypes: true });
      
      for (const entry of entries) {
        const entryPath = path.join(dirPath, entry.name);
        
        if (entry.isDirectory()) {
          size += await this.getDirSize(entryPath);
        } else {
          const stat = await fs.stat(entryPath);
          size += stat.size;
        }
      }
    } catch {
      // Directory doesn't exist
    }
    
    return size;
  }
  
  async countFiles(dirPath) {
    let count = 0;
    
    try {
      const entries = await fs.readdir(dirPath, { withFileTypes: true });
      
      for (const entry of entries) {
        const entryPath = path.join(dirPath, entry.name);
        
        if (entry.isDirectory()) {
          count += await this.countFiles(entryPath);
        } else {
          count++;
        }
      }
    } catch {
      // Directory doesn't exist
    }
    
    return count;
  }
  
  async getMetadata(userId) {
    const metadataPath = path.join(this.getUserDir(userId), '.metadata');
    
    try {
      const data = await fs.readFile(metadataPath, 'utf-8');
      return JSON.parse(data);
    } catch {
      return { last_access: null };
    }
  }
  
  async updateMetadata(userId) {
    const metadataPath = path.join(this.getUserDir(userId), '.metadata');
    
    const metadata = await this.getMetadata(userId);
    metadata.last_access = new Date().toISOString();
    
    await fs.mkdir(path.dirname(metadataPath), { recursive: true });
    await fs.writeFile(metadataPath, JSON.stringify(metadata, null, 2));
  }
  
  async checkStorage() {
    try {
      await fs.access(this.userDataRoot);
      return true;
    } catch {
      return false;
    }
  }
}

module.exports = { storageService: new StorageService() };
```

### src/middleware/auth.js

```javascript
const jwt = require('jsonwebtoken');
const fs = require('fs');
const path = require('path');
const { config } = require('../config');
const { logger } = require('../utils/logger');

// Load API keys from file
let apiKeys = [];

function loadApiKeys() {
  const keysFile = process.env.ZERBOOT_API_KEYS_FILE || '/var/lib/zeroboot/api_keys.json';
  
  try {
    const data = fs.readFileSync(keysFile, 'utf-8');
    apiKeys = JSON.parse(data);
    logger.info(`Loaded ${apiKeys.length} API keys`);
  } catch (error) {
    logger.warn(`Failed to load API keys: ${error.message}`);
    apiKeys = [];
  }
}

loadApiKeys();

// API Key middleware
function apiKeyMiddleware(req, res, next) {
  const apiKey = req.get(config.security.apiKeyHeader);
  
  if (!apiKey) {
    // Try JWT token
    return jwtMiddleware(req, res, next);
  }
  
  const keyConfig = apiKeys.find(k => k.key === apiKey);
  
  if (!keyConfig) {
    return res.status(401).json({
      success: false,
      error: 'Invalid API key',
      code: 'INVALID_API_KEY',
    });
  }
  
  req.user = {
    user_id: keyConfig.user_id,
    rate_limit: keyConfig.rate_limit,
    auth_type: 'api_key',
  };
  
  next();
}

// JWT middleware (for future use)
function jwtMiddleware(req, res, next) {
  const authHeader = req.get('Authorization');
  
  if (!authHeader?.startsWith('Bearer ')) {
    return res.status(401).json({
      success: false,
      error: 'Missing authentication',
      code: 'AUTH_REQUIRED',
    });
  }
  
  const token = authHeader.substring(7);
  
  try {
    const decoded = jwt.verify(token, config.security.jwtSecret);
    req.user = {
      user_id: decoded.sub,
      rate_limit: decoded.rate_limit,
      auth_type: 'jwt',
    };
    next();
  } catch (error) {
    return res.status(401).json({
      success: false,
      error: 'Invalid token',
      code: 'INVALID_TOKEN',
    });
  }
}

// Generic auth middleware (accepts either)
function authMiddleware(req, res, next) {
  apiKeyMiddleware(req, res, next);
}

module.exports = {
  authMiddleware,
  apiKeyMiddleware,
  jwtMiddleware,
};
```

```javascript
const rateLimit = require('express-rate-limit');
const { config } = require('../config');
const { logger } = require('../utils/logger');

// Key generator for rate limiting
function keyGenerator(req) {
  const userId = req.user?.user_id || req.ip;
  return `rl_${userId}`;
}

// Rate limit exceeded handler
function rateLimitHandler(req, res) {
  logger.warn(`Rate limit exceeded`, {
    userId: req.user?.user_id,
    ip: req.ip,
  });
  
  res.status(429).json({
    success: false,
    error: 'Rate limit exceeded',
    code: 'RATE_LIMIT_EXCEEDED',
    details: `Limit: ${config.rateLimit.maxRequests} requests per ${config.rateLimit.windowMs / 1000} seconds`,
  });
}

// Custom skip function for certain paths
function skipIfAdmin(req) {
  return req.user?.rate_limit > 1000;
}

// Rate limit middleware
function rateLimitMiddleware(req, res, next) {
  const limiter = rateLimit({
    windowMs: config.rateLimit.windowMs,
    max: req.user?.rate_limit || config.rateLimit.maxRequests,
    keyGenerator,
    handler: rateLimitHandler,
    skip: skipIfAdmin,
    standardHeaders: true,
    legacyHeaders: false,
  });
  
  limiter(req, res, next);
}

module.exports = {
  rateLimitMiddleware,
};
```

### src/utils/security.js

```javascript
const { config } = require('../config');

// Forbidden patterns for code validation
const FORBIDDEN_PATTERNS = [
  // Dangerous file operations
  { pattern: /rm\s+-rf\s+\/(?!workspace)/, message: 'Root directory deletion forbidden' },
  { pattern: /rm\s+-rf\s+\/home/, message: 'Home directory deletion forbidden' },
  
  // Disk operations
  { pattern: /dd\s+if=/, message: 'Direct disk operations forbidden' },
  { pattern: /mkfs\./, message: 'Filesystem formatting forbidden' },
  
  // Fork bombs
  { pattern: /:(\s)*\(\s*\)\s*\{\s*:\|:&\s*\};\s*:/, message: 'Fork bomb detected' },
  
  // Remote code execution
  { pattern: /curl\s+.+\|sh/, message: 'Pipe to shell forbidden' },
  { pattern: /wget\s+.+\|sh/, message: 'Pipe to shell forbidden' },
  
  // Network scanning
  { pattern: /nmap\s+/, message: 'Network scanning forbidden' },
  { pattern: /nc\s+-l\s+/, message: 'Network listeners forbidden' },
  
  // Privilege escalation
  { pattern: /sudo\s+/, message: 'sudo forbidden' },
  { pattern: /chmod\s+[47]777/, message: 'Dangerous permissions forbidden' },
];

function validateCode(code) {
  // Check size
  if (code.length > config.security.maxCodeSize) {
    const error = new Error(`Code exceeds maximum size of ${config.security.maxCodeSize} bytes`);
    error.status = 400;
    error.code = 'CODE_TOO_LARGE';
    throw error;
  }
  
  // Check patterns
  for (const { pattern, message } of FORBIDDEN_PATTERNS) {
    if (pattern.test(code)) {
      const error = new Error(message);
      error.status = 400;
      error.code = 'FORBIDDEN_CODE';
      throw error;
    }
  }
  
  // Check for null bytes
  if (code.includes('\0')) {
    const error = new Error('Null bytes not allowed');
    error.status = 400;
    error.code = 'INVALID_CODE';
    throw error;
  }
  
  return true;
}

function sanitizePath(userPath, basePath) {
  // Normalize and resolve path
  const resolved = path.resolve(basePath, userPath);
  
  // Ensure it's within base path
  if (!resolved.startsWith(basePath)) {
    const error = new Error('Path traversal detected');
    error.status = 400;
    error.code = 'INVALID_PATH';
    throw error;
  }
  
  return resolved;
}

module.exports = {
  validateCode,
  sanitizePath,
};
```

## 11. 与AgentChat集成

### 11.1 AgentChat环境变量配置

在AgentChat服务器的 `.env.local` 中添加：

```bash
# 启用沙盒功能
SANDBOX_ENABLED=true

# Gateway服务地址（内网地址）
SANDBOX_GATEWAY_URL=http://sandbox-server-internal-ip:8080

# API密钥（与Gateway配置一致）
SANDBOX_API_KEY=your-secure-api-key
```

### 11.2 工具使用说明

启用沙盒后，Agent将自动获得以下工具：

| 工具名称 | 功能 | 示例 |
|---------|------|------|
| bash | 执行shell命令 | `ls -la` |
| readFile | 读取文件 | 读取 `config.json` |
| writeFile | 写入文件 | 创建 `output.txt` |

### 11.3 数据隔离

- 每个用户的数据存储在独立的目录中
- 用户ID经过SHA256哈希后作为目录名
- 不同用户之间完全隔离，无法互相访问

### 11.4 会话管理

- 每个对话对应一个独立的会话（sessionId = conversationId）
- 会话闲置30分钟后自动销毁VM（数据保留）
- 再次访问时自动重新激活

## 12. AgentChat故障排查

### 常见问题

1. **沙盒工具不可用**
   - 检查 `SANDBOX_ENABLED=true`
   - 检查Gateway服务是否运行
   - 检查API Key是否正确

2. **命令执行失败**
   - 检查Zeroboot服务状态
   - 查看Gateway日志: `journalctl -u sandbox-gateway -f`

3. **用户数据丢失**
   - 检查 `/var/lib/zeroboot/users` 权限
   - 确保磁盘空间充足
